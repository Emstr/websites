# Camden Associates - Corporate Finance House

A sophisticated corporate finance website built for Camden Associates, a London-based corporate finance advisory specializing in international deal-making.

## Business Profile
- **Business**: Camden Associates
- **Website**: camdenassociates.co.uk
- **Location**: London
- **Specialization**: Corporate Finance
  - Biotech
  - Technology
  - Real Estate
  - Junior Mining & Exploration

## Recent Clients
- THERAVECTYS (Paris biotech - vaccine platform)
- Mooveco-Muses (French startup - last mile delivery)

## Key Features
- **Dark, Premium Aesthetic**: Sophisticated dark mode design with electric blue accents
- **Prominent Contact Information**: Addressing the main issue of their current site
- **Sector-Focused**: Dedicated pages for each specialty area
- **International Feel**: Designed for cross-border European deals
- **Case Study Highlights**: Showcasing recent successful transactions

## Tech Stack
- Next.js 16+
- React 19
- TypeScript
- Tailwind CSS v4
- Vercel Analytics

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Color Scheme
- **Dark Charcoal**: #1a1a2e (Primary background)
- **Electric Blue**: #4361ee (Primary accent)
- **Silver/White**: #e8e8e8 / #ffffff (Text and highlights)

## Design Philosophy
Premium, deal-focused corporate finance aesthetic with emphasis on:
- International credibility
- Sector expertise
- Accessible contact information
- Professional sophistication
