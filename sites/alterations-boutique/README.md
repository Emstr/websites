## Services

- **Men's Alterations**: Suits, trousers, jackets, shirts
- **Women's Alterations**: Dresses, skirts, trousers, blouses
- **Bridal Alterations**: Wedding dresses, bridesmaid dresses
- **Express Service**: Same-day and urgent alterations
- **Call-Out Service**: Mobile fitting and consultation
- **Free Consultation**: Expert advice and fitting at no charge

## Tech Stack

- **Framework**: Next.js 15+
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Font**: Geist Sans
- **Analytics**: Vercel Analytics

## Color Scheme

- **Burgundy**: #881337 (primary brand color)
- **Silver**: #94a3b8 (secondary/accent)
- **White**: Background and text contrast

## Design Philosophy

Professional, accessible, and multi-location business feel. More accessible than luxury exclusive tailors - emphasizing reliability, experience, and comprehensive service across three locations.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
