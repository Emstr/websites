- **Framework:** Next.js 16+

- **Language:** TypeScript
- **Font:** Inter (Google Fonts)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Brand Positioning

The London Flower Shop is positioned as a traditional, warm, family-run florist that emphasizes:

- **Community Connection:** Deep relationships with local businesses and residents
- **Family Values:** Genuine family business with personal touch
- **Reliability:** Trusted partner for regular customers and special events
- **Versatility:** Wide product range beyond just flowers
- **Accessibility:** Friendly, approachable service for everyone
- **Heritage:** 20+ years as a florist at this location, family-run since 2009

This contrasts with modern, trendy competitors by focusing on warmth, tradition, and community bonds.
