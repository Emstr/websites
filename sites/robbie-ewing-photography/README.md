## About the Business

Robbie Ewing is a professional commercial portrait photographer based in Camberwell, South London. With a fully equipped studio and a creative approach to portrait photography, Robbie specializes in capturing the unique features and personality of each subject.

**Key Services:**
- **Corporate Headshots**: Professional headshots for business professionals, executives, and teams
- **Creative Portraits**: Artistic portrait sessions with unique lighting and composition
- **Musicians & DJs**: Dynamic portraits for performing artists and press kits
- **Model Portfolios**: Comprehensive portfolio shoots showcasing range and versatility
- **Commercial Work**: High-quality commercial photography for businesses and brands

**Approach**: Professional portrait photographer specializing in unique creative work that captures the best features of each subject. Every session is personalized to discover and highlight what makes each individual distinctive.

**Studio**: Fully equipped photography studio in Camberwell, South London, with professional lighting, versatile backdrops, and a comfortable environment designed for exceptional portrait photography.

## Website Design

**Theme**: Minimal Photography Portfolio / Gallery Aesthetic

**Color Palette**:
- Pure Black: `#000000` (primary background)
- White: `#ffffff` (text and highlights)
- Accent Red: `#ef4444` (call-to-action and accents)
- Grays: Various shades for depth and hierarchy

**Design Philosophy**:
- Visual-first approach - let the work speak
- Minimal, clean typography
- Large image placeholders for portfolio pieces
- Gallery-style layouts
- Professional but artistic aesthetic
- Black background for gallery/photographer feel
- Red accent for strategic emphasis
- Mobile-responsive design

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Build for production:
```bash
npm run build
npm start
```
