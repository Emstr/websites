# Hirschfelds Jewellery Repairs

A heritage luxury website for Hirschfelds, one of Britain's oldest jewellery repair workshops, established in Hatton Garden since 1875.

## Business Information

- **Business**: Hirschfelds (Jewellery Repairs London)
- **Phone**: 020 7405 1536
- **Location**: Hatton Garden, London (UK's historic jewellery quarter)
- **Founded**: 1875 (150 YEARS of heritage!)
- **Hours**: Monday-Friday 11am-4pm (by appointment only)
- **Services**: Jewellery Repairs, Ring Sizing, Bespoke Manufacturing, Restoration
- **USP**: One of the country's oldest jewellery repair workshops with 150 years of master craftsmanship

## Getting Started

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
npm run start
```
