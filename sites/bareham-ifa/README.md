# Bareham IFA - Independent Financial Advisers Leeds

A Next.js demo website for Bareham IFA, a family-owned independent financial advisory firm in Leeds.

## Business Details

- **Name**: Bareham IFA
- **Website**: barehamifa.com
- **Email**: info@barehamifa.com
- **Address**: Foxborough House, 96 Allerton Grange Rise, Leeds LS17 6LG
- **FCA Reference**: 939595
- **Founded**: 1985 by Gordon Bareham
- **Current Director**: James Bareham (son of Gordon)

## Services

- Investment Planning
- Pension Planning
- Retirement Planning

## Unique Selling Points

1. **Family Succession Story**: From Gordon to James Bareham - three generations of trusted financial guidance
2. **Independent & Bespoke**: Family-owned independence means no corporate pressures, just honest advice
3. **Plain English Approach**: Financial planning explained clearly without jargon - "the Yorkshire way"
4. **Local Leeds Expertise**: Based in Allerton, serving families across Leeds and Yorkshire

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
