# Courtneys Estate Agents - Dalston

Demo website for Courtneys Estate Agents, a family-run estate agency serving Hackney, East & North London since 2003.

## Business Information

- **Business**: Courtneys Estate Agents
- **Location**: 544 Kingsland Rd, London E8 4AH
- **Phone**: 0207 275 8000
- **Founded**: 2003 (20+ years of service)
- **Type**: Family owned and operated

## Services

- Residential Sales
- Residential Lettings & Management
- Commercial Sales & Lettings
- Mortgages
- Land & Development
- Property Search

## Running the Site

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Technology Stack

- Next.js 16+
- React 19
- Tailwind CSS 4
- TypeScript
