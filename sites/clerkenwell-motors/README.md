# Clerkenwell Motors

Demo website for **Clerkenwell Motors** - an independent garage in Central London.

## Business Information

- **Name:** Clerkenwell Motors
- **Location:** 14 Warner Street, London EC1R 5HA
- **Phone:** 020 7278 6375
- **Email:** info@clerkenwellmotors.co.uk
- **Established:** 1993 (30+ years family-owned)
- **Rating:** 4.7/5 on Google (111 reviews)

## Services

- MOT Testing
- Full Vehicle Servicing
- Mechanical Repairs
- Car Diagnostics
- Tyres & Wheel Services
- Air Conditioning Service
- Body Repairs & Cosmetics
- Performance Tuning
- Electronics & Auto Electrical

## Operating Hours

- **Monday - Friday:** 7:30 - 18:30
- **Saturday:** 9:00 - 14:00 (by appointment)
- **Sunday:** Closed

## Accreditations

- RAC Approved Garage
- AA Approved Garage
- MIPOC Member
- Which? Trusted Trader

## Coverage Areas

Serving Islington, Camden, The City, Clerkenwell, King's Cross, Angel, Farringdon, and surrounding Central London areas.

## Vehicle Types

- Cars (all makes and models)
- Vans
- Motorcycles
- Classic vehicles
- Electric and hybrid vehicles

## Technology Stack

- **Framework:** Next.js 16+
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Fonts:** Inter (Google Fonts)

## Design

Unique automotive garage template featuring:
- Deep red (#b91c1c) and charcoal (#1f2937) color scheme
- Professional, trust-building aesthetic
- Mobile-responsive design
- Emphasis on 30+ years family ownership
- Prominent display of accreditations and 4.7/5 rating

## Pages

1. **Home** - Hero section, services overview, why choose us
2. **Services** - Comprehensive list of all services offered
3. **MOT** - Dedicated MOT testing information
4. **About** - Company history, accreditations, why choose us
5. **Reviews** - Customer testimonials and Google rating
6. **Contact** - Contact details, opening hours, location map

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
