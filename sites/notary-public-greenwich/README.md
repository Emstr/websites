# Notary Public Islington

Professional website for Esther Ogun, Notary Public based in Islington, London EC1.

## Overview

A modern, responsive website built with Next.js 16 and Tailwind CSS, featuring:

- **Homepage** - Hero section, services overview, pricing, and areas served
- **Business Services** - Corporate documentation, commercial contracts, apostille services
- **Individual Services** - Personal document certification, powers of attorney, travel permissions
- **About** - Qualifications, approach, and fee information
- **Contact** - Contact form, phone numbers, email, and office location

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Geist font family

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

## Deploy

Deploy to Vercel:

```bash
vercel
```
