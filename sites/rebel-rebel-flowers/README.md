## Design Concept

This website captures Rebel Rebel's bold, rebellious aesthetic:

### Color Palette

- **Hot Pink**: #ec4899 (rebel-pink)
- **Deep Purple**: #7c3aed (rebel-purple)
- **Black**: #000000 (rebel-black)
- **Dark Grey**: #0a0a0a, #1a1a1a (rebel-dark, rebel-grey)

### Design Philosophy

- **Bold & Edgy**: NOT a typical "pretty" florist site
- **Dark Background**: Black and dark grey with bold color pops
- **Gradient Accents**: Pink-to-purple gradients throughout
- **A-List Focus**: Prominently features high-profile clients
- **Rebellious Messaging**: "Flagrant disregard for the colour wheel"
- **25 Years Legacy**: Emphasizes long history and expertise

## Technical Stack

- **Framework**: Next.js 16+ with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**:
  - Inter (body text)
  - Playfair Display (display/headings)

## Site Structure

6. **Contact** (`/contact`) - Business details, FAQ, opening hours

### Components

- **Navigation**: Sticky header with gradient CTA button
- **Footer**: Multi-column with brand, links, contact, credentials

## Getting Started

### Prerequisites

- Node.js 20.x or higher
- npm or yarn

### Installation

```bash
# Navigate to project directory
cd /home/user/websites/sites/rebel-rebel-flowers

# Install dependencies
npm install

# Run development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
# Create production build
npm run build

# Start production server
npm start
```

## Development Notes

### Color System

Custom Tailwind colors defined in `tailwind.config.js`:
- `rebel-pink`: Hot pink (#ec4899)
- `rebel-purple`: Deep purple (#7c3aed)
- `rebel-black`: Pure black (#000000)
- `rebel-dark`: Very dark grey (#0a0a0a)
- `rebel-grey`: Dark grey (#1a1a1a)

### Custom Utilities

Defined in `globals.css`:
- `.text-gradient-rebel`: Pink-to-purple gradient text
- `.border-gradient-rebel`: Pink-to-purple gradient borders

### Typography

- Body: Inter (sans-serif)
- Display/Headings: Playfair Display (serif)
