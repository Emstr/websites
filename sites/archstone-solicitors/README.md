## Tech Stack

- Next.js 15.1.0
- React 19
- TypeScript
- Tailwind CSS
- Geist Font

## Design

- **Colors:** Deep navy (#1a365d), Gold accents (#d69e2e), White
- **Style:** Professional law firm aesthetic with scales of justice theme
- **Mobile Responsive:** Yes
- **SEO Optimized:** Yes with JSON-LD structured data

## Pages

- Home - Hero section, practice areas overview, testimonials, location info
- Practice Areas - Detailed information about all legal services
- Our Team - Solicitor profiles and qualifications
- Testimonials - Client reviews and ratings
- Fees - Transparent pricing information
- Contact - Contact form and office information
- Privacy Policy
- Terms of Service

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```

## Key Features

- Unique professional law firm design (different from notary sites)
- Practice area cards with legal icons
- Team member profiles
- Client testimonials
- Fee transparency section
- Location highlights (near Westfield Stratford)
